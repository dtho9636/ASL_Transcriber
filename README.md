# Sign Language Transcriber

## Overview

This project aims to build a real-time sign language transcription system using a webcam. The program will capture video input, detect hand movements, and translate sign language into readable text.

## Goals

* Capture live webcam video
* Detect and track hands
* Recognize sign language gestures
* Output translated text in real time

## Tech Stack

* Python
* OpenCV (webcam + image processing)
* MediaPipe (hand tracking)
* Machine learning model (TensorFlow or PyTorch)
